# Configuration de Proxmox
### Après avoir installé Proxmox, il est temps de se connecter sur l'interface web !  
![](2.co.png)  

Depuis un navigateur tapez l'adresse IP du serveur Proxmox ( il faut être dans le même range IP ou avoir des règles de flux qui autorise la connection sur cette IP)  
![](2.1seco.png)  
Proxmox utilise un certificat autosigné d'où le message d'erreur. Il faut cliquer sur "Continuez vers..."
